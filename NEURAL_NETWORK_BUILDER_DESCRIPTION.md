## Technical Description (150 words)
A client-side machine learning playground built with Next.js 14, React 19, TypeScript, and TensorFlow.js. Features real-time neural network training with live visualization of training metrics, interactive scatter plots with decision boundary rendering, and dynamic neuron contribution analysis based on weight magnitudes.

The tool provides a complete educational experience including:
- Pre-configured network templates (beginner to advanced)
- Customizable hyperparameters (activation functions, optimizers, loss functions)
- Real-time training progress with dual-axis charts (loss & accuracy)
- SVG-based network visualization with contribution-based gradient coloring
- Interactive testing interface with probability distribution displays
- Achievement system with localStorage persistence
- Responsive design with mobile-first approach
- SSR-compatible with proper hydration handling

All computations run client-side using TensorFlow.js, ensuring privacy and instant feedback without server dependencies.
